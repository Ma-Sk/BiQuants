Далее учитываются только значащие колонки т.е. начиная с ACT_INT__POLITICAL_VIEWING_ON_TV___LIBERAL_COMEDY_14823

в файлах %сегмент%percent указано процентное соотношение кол-ва пустых в колонке относительно всех записей в сегменте

в файлах %сегмент%amount указано кол-ва пустых в колонке

в файлах %сегмент%Nulls-RowCount сколько записей имело то или иное кол-во пустых значений(первое - кол-во пустых в записи, второе - кол-во таких записей)

в файлах %сегмент%nullCount-percent кол-во записей имеющих хоть один null и процентное отношение пустых записей ко всем записям

в файлах %сегмент%nullColls указаны имена колонок в которых встречались пустые значения
